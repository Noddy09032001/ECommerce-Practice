# ECommerce-Application

A full-stack e-commerce application built with modern technologies, focusing on secure authentication, payment processing, role-based access control, and real-world order management workflows.

Backend
Java 21
Spring Boot
Spring Security
Spring Data JPA
MySQL
JWT Authentication
Maven
Frontend
React.js
Next.js
TypeScript
Tailwind CSS
Third-Party Services
Stripe Payment Gateway
Email Service (SMTP)
In-App Notification System
